Guessing Game


How to play this game?


When you open or reload this page,you can see a prompt box.You should input number between 1 to 5.


You can get 05 chances to play this game.I mean you can guess 5 numbers.You write a number then press ok.then Again write a number then press ok.Thus you can input number for 5 times.


After write 5 numbers you can show correct/wrong answer in console.


Atlast,Press "Show result" button to see your points.


1 correct answer for 10 points.Total points 50.
